# QIDI Q2 - Оптимизированные конфигурации (108-111)

## 📋 Описание

Оптимизированные конфигурационные файлы для 3D-принтера **QIDI Q2** с прошивками версий **108-111**. Конфигурация разработана Виктором Волковым (@Volvik88) и включает множество улучшений по сравнению со стоковыми конфигами.

> ⚠️ **ВАЖНО:** Конфигурации **НЕ РАБОТАЮТ** с прошивкой старше версии 111!  
> Для прошивки 1122 используйте [другую версию](https://wiki.qidi-russia.ru/).

## 🚀 Основные улучшения

### 🔧 Функциональность:
1. ✅ **Починен адаптивный автоуровень** - принтер берёт пробы только в области печати модели
2. ⚡ **Увеличены скорости** перемещения при автодоме и автоуровне
3.  **Полностью переписан стартовый код** принтера
4. ✂️ **Автоматическая обрезка филамента** при отмене и окончании печати
5. 💨 **Автоматическое включение вытяжки** с фильтром после печати
6. 🌬️ **Стряхивание и охлаждение сопла** после загрузки прутка
7. 🌡️ **Предварительный нагрев сопла** у стартовой полосы перед печатью
8. 🧹 **Оптимизирована очистка сопла** (протирка, щётка)
9. 🔊 **Звуковые сигналы** для важных событий
10. 🎯 **Умная парковка** для избежания подтёков

### 🖨️ Печать:
- Уменьшены объёмы и скорость прочистки сопла
- Отходы сгруппированы и меньше застревают
- Корректная работа с **QIDI BOX**
- Поддержка **высокотемпературных пластиков**
- Два режима окончания печати: **мягкий** и **быстрый**
- Ретракт и быстрые действия при окончании

### 🖥️ Интерфейс:
- Включение вытяжки **не паузит дисплей**
- Увеличено время простоя паузы **до 2 суток**
- Описания к макросам (подсказки при наведении)
- Макрос **PREHEAT** для предпрогрева с таймерами

### 🛡️ Безопасность:
- Защита реверса принтера (от заводского брака команды M4031)
- Исправлен температурный баг при окончании филамента
- Автоматическое отключение светодиодов и моторов в простое
- Автоматическое включение датчика филамента

  ### Основные компоненты:
- **KAMP** (Klipper Adaptive Meshing & Purging) - адаптивная сетка и очистка
- **Adaptive_Mesh.cfg** - динамическая калибровка стола
- **KAMP_Settings.cfg** - настройки KAMP
- **Moonraker.conf** - конфигурация API сервера
- **Crowsnest.conf** - настройка веб-камеры
- **FluidD.cfg** - макросы и улучшения интерфейса

### Конфигурационные файлы:
- `printer.cfg` - основной файл конфигурации
- `mcu_id.cfg` - настройки MCU
- `box*.cfg` - конфигурации коробок (1-4)
- `gcode_macro.cfg` - пользовательские макросы
- `plc.sh` - скрипты
- `timelapse.cfg` - настройка таймлапсов
- `drying.conf` - сушка филамента
- `saved_variables.cfg` - сохранение переменных
